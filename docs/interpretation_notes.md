# 结果解释边界

- 本研究不应被写成“高准确率焦虑检测器”。
- 主结论是：在严格 subject-level、弱标签、小样本和跨数据集域偏移条件下，EEG 全局频谱组织保留了弱但可检查的焦虑相关信号。
- JointConstraintNet 的定位是轻量稳定化结构，不是高容量深度网络的性能竞赛。
- Mendeley 缺少部分训练特征，只能作为 partial-overlap compatibility test。
- ds007216 结果用于分析 domain-direction mismatch；反向判别不是模型可直接泛化的证据。
- smoke fixture 是合成数据，只能证明代码可跑，不能报告为论文实验结果。
